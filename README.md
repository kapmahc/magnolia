# Magnolia - A complete open source e-commerce solution for Grails.

## Environment to prepare
* Install grails
```bash
curl -s "https://get.sdkman.io" | bash
source "$HOME/.sdkman/bin/sdkman-init.sh"
sdk install java
sdk install gradle
sdk install grails
grails -version
```

## Development
```bash
git clone https://github.com/kapmahc/magnolia.git
cd magnolia
grails run-app
```
